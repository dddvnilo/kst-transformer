# Setup

## 1. Creating venv

### Windows

```bash
python -m venv venv
```

### Linux / macOS

```bash
python3 -m venv venv
```


## 2. Activating venv

### Windows PowerShell

```powershell
.\venv\Scripts\Activate.ps1
```

### Windows CMD

```cmd
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## 3. Installing dependencies

```bash
pip install -r requirements.txt
```
